# Fake News Detector using BERT

## Overview

This project is a Fake News Detection system built using Natural Language Processing (NLP) and the BERT Transformer model.

The model classifies news articles as either REAL or FAKE using machine learning and deep learning techniques.

---

## Technologies Used

* Python
* BERT Transformer
* PyTorch
* Scikit-learn
* Pandas
* Google Colab

---

## Features

* Fake news classification
* TF-IDF + Logistic Regression baseline model
* BERT Transformer fine-tuning
* News prediction system
* Model saving and loading

---

## Dataset

Dataset used:

* Fake and Real News Dataset from Kaggle

Files:

* Fake.csv
* True.csv

---

## Workflow

1. Data Collection
2. Data Preprocessing
3. TF-IDF Vectorization
4. Logistic Regression Classification
5. BERT Tokenization
6. BERT Fine-Tuning
7. Prediction and Evaluation
